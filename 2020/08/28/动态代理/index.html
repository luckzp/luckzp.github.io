动态代理

发表于 2020-08-28 更新于 2021-06-14 阅读次数：

JAVA 中的动态代理：通过动态的创建一个新类，实现在运行时对类进行修改（如想知道某个类运行的时长）。

使用 JDK 和 Cglib 实现动态代理。

Spring 如何选择两种代理模式的？

1、如果目标对象实现了接口，则默认采用 JDK 动态代理。

2、如果目标对象没有实现接口，则使用 Cglib 代理。

我们可以通过 DeBug 看到生成的代理类后缀不同来区别。
